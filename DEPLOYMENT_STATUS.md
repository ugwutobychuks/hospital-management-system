## 🎯 Deployment Options

### 1. Local Development
```bash
docker-compose up -d
npm run dev
```

### 2. Docker
```bash
docker build -t hms:latest .
docker run -p 3000:3000 hms:latest
```

### 3. Kubernetes
```bash
kubectl apply -f k8s/deployment.yaml
kubectl apply -f k8s/scaling.yaml
```

### 4. Automated Deployment
```bash
chmod +x deploy.sh
./deploy.sh
```
